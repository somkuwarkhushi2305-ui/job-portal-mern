# 💼 Job Portal (MERN Stack)

A full-stack Job Portal web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The platform allows recruiters to post jobs and manage applicants, while students can browse jobs, apply, and track their application status.

---

## 🚀 Features

### 👨‍🎓 Student
- User Registration & Login
- Browse Available Jobs
- Search & Filter Jobs
- View Job Details
- Apply for Jobs
- Track Application Status (Pending / Accepted / Rejected)
- Update Profile
- Upload Resume
- Save Jobs for Later

### 👨‍💼 Recruiter
- Recruiter Registration & Login
- Create Company Profile
- Upload Company Logo
- Post New Jobs
- View Posted Jobs
- View Applicants
- Accept / Reject Applications

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/somkuwarkhushi2305-ui/job-portal-mern.git
```

### Backend

```bash
cd Backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
